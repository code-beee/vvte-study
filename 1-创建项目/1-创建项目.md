# 创建项目

## 🎯 目标

- 创建项目。
- 项目最小化，我需要一张白纸。

## 🍸 准备

### 开发环境

| 名称    | 版本      |
| ------- | --------- |
| Node.js | `14.17.3` |
| NPM     | `6.14.13` |

> 💡 以上是我的本地环境。

### 初始化Vite

> 执行初始化命令后会有提示，按提示操作即可（上下键选择、回车确认）。

打开 CMD ，输入 vite 初始化命令：

```bash
npm init @vitejs/app
```

控制台输出内容如下：

```text
√ Project name: ... vvte
√ Select a framework: » vue
√ Select a variant: » vue-ts

Scaffolding project in F:\WorkSpace\bee-code\bee-dev-ui...

Done. Now run:

  cd vvte
  npm install
  npm run dev
```

### 启动项目

```bash
# 进入项目目录
cd vvte
# 安装依赖
npm install
# 在dev模式下运行
npm run dev
```

> 在浏览器输入 http://localhost:3000 访问。

## 🌈 Coding

### 精简 HelloWorld.vue

代码如下：

```vue
<template>
  <h2>Hello World!</h2>
</template>

<script lang="ts">
import { defineComponent } from "vue";

export default defineComponent({
  name: "HelloWorld",
  setup: () => {},
});
</script>
```

### 精简 App.vue

代码如下：

```vue
<template>
  <HelloWorld />
</template>

<script lang="ts">
import { defineComponent } from "vue";
import HelloWorld from "./components/HelloWorld.vue";

export default defineComponent({
  name: "App",
  components: {
    HelloWorld,
  },
});
</script>

```

> 💡 静态文件 `/src/assets/logo.png` 和`/public/favicon.icon` 也可以删除。看你心情了。
