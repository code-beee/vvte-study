# 创建项目

## 开发环境

| 名称    | 版本      |
| ------- | --------- |
| Node.js | `14.15.1` |
| NPM     | `6.14.8`  |

## 初始化

> 执行初始化命令后会有提示，按提示操作即可（上下键选择、回车确认）。

打开 CMD ，输入 vite 初始化命令：

```bash
npm init @vitejs/app
```

控制台输出内容如下：

```text
√ Project name: ... vvte
√ Select a framework: » vue
√ Select a variant: » vue-ts

Scaffolding project in F:\WorkSpace\bee-code\bee-dev-ui...

Done. Now run:

  cd vvte
  npm install
  npm run dev
```

## 启动项目

```bash
# 进入项目目录
cd vvte
# 安装依赖
npm install
# 在dev模式下运行
npm run dev
```

在浏览器输入 http://localhost:3000 访问。
